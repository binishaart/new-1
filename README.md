Features

Simplifies complex text using AI

Works with any science, legal, or technical paragraph

Outputs a clear, child-friendly explanation

Requirements

Python 3.x (3.10+ recommended)

OpenAI Python library

Install the library using pip:

pip install openai

Setup

Clone or download this repository.

Open the Python file eli5_tool.py.

Replace "YOUR_API_KEY_HERE" with your OpenAI API key.

Add your complex paragraph in the complex_paragraph variable.
